# Accumulators ⚡
A fullstack web application for generating and analyzing accumulator bets with a clean UI and data-driven backend.

## 🚀 Overview
Accumulators is a web app designed to help users create, manage, and analyze accumulator bets efficiently.  
It combines a responsive frontend with a powerful backend to process and evaluate betting data.

This project demonstrates fullstack development skills, API design, and data handling.

---

## ✨ Features
- Create and manage accumulator entries
- Dynamic frontend with interactive UI
- Backend API for processing and storing data
- Data handling using Python (Pandas/NumPy if applicable)
- Persistent storage using PostgreSQL
- Clean and modular architecture

---

## 🛠 Tech Stack
**Frontend**
- React
- JavaScript (ES6)
- HTML / CSS

**Backend**
- FastAPI (Python)

**Database**
- PostgreSQL

**Data Tools**
- Pandas / NumPy
